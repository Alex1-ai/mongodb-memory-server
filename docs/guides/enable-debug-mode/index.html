Enable Debug Mode

The Debug Mode can be enabled by setting the DEBUG config options, by either setting it as a Environment Variable or package.json option.

Examples#

Via Environment Variable#

# also available case-insensitive values: "on" "yes" "true"MONGOMS_DEBUG=1 npm run command

Via package.json config#

{  "config": {    "mongodbMemoryServer": {      "debug": "1", // also available case-insensitive values: "on" "yes" "true"    }  }}
